# AGENTS.md - Notes Repository

This is a personal knowledge base containing technical notes about systems programming.

## Structure

- Root directory: Miscellaneous notes on CPU architecture, OS concepts, algorithms, etc.
- `ai_gen/`: AI-assisted technical documentation
- `riscv/`: RISC-V architecture notes
- `cpu硬件/`: CPU microarchitecture studies
- `ARM/`: ARM architecture and virtualization
- `virt/`: Virtualization topics including migration

## Usage

This is a reference/memo repository, not a software project. No build, test, or CI workflows exist.

When editing notes:
- Preserve technical accuracy
- Keep entries concise and focused
- Use Chinese headings/explanations as appropriate (most existing notes are in Chinese)
